How to use halo
  1. Select a .json or .json.gz halo file in the Input tab.
  2. Click Launch Analysis to load and render the plots.
  3. Switch to Results to explore by sample or chromosome, zoom, and sync charts.
  4. Use Show Example to load the bundled demo dataset if you don’t have a file handy.

Example dataset: HG00733.6-samples.json.gz

Visit the halo command-line application for usage instructions on how to create the .json input file.
